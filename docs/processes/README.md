### 1. [시스템 프로세스 가이드](./PROCESS.md)

- 요청 처리 프로세스
- 데이터 흐름
- 서버 시작 프로세스
- 에러 처리 프로세스

### 2. [환경 설정 프로세스](./SETUP_ENV.md)
- .env 파일 생성 방법
- Notion API Key 발급

- Integration 연결 방법

### 3. [배포 프로세스](./DEPLOYMENT.md)
- Kubernetes 배포 가이드

### 4. [트러블슈팅 프로세스](./TROUBLESHOOTING.md)

- 빠른 진단 체크리스트
- 긴급 롤백 방법

## 🚀 빠른 시작

### 처음 시작하는 경우
1. [환경 설정 프로세스](./SETUP_ENV.md)를 따라 환경을 설정하세요
2. [시스템 프로세스 가이드](./PROCESS.md)를 읽어 시스템 구조를 이해하세요

### 배포하는 경우
1. [배포 프로세스](./DEPLOYMENT.md)를 참고하세요
2. 문제가 발생하면 [트러블슈팅 프로세스](./TROUBLESHOOTING.md)를 확인하세요
